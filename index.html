Boilerplait

Boilerplait helps you get started with your react and nodejs app quickly. It helps you spin off a new app with a few simple commands so that you can focus on your application logic rather than worrying about the logistics that go into starting a front/backend app. Boilerplait come in handy in cases where you want to test a new component and you are not certain that the component will fit your needs; in these cases you clone boilerplait and start testing the new component to get to your results faster.

Usage

There is a lot of basic setup instructions provided here, please skip the steps that you don't need (e.g. you don't have to install webpack if you have already installed it globally).

  1. Install npm: sudo apt-get install npm // if this doesn't work the try the next line curl https://www.npmjs.org/install.sh | sudo sh +. Install node: sudo apt-get update sudo apt-get install node +. npm install express -g +. npm install -g express-generator +. Restart your terminal to pick up the latest express +. Install webpack: npm install webpack -g +. git clone https://github.com/wmf23/boilerplait.git +. npm install +. Start webpack with watch in the background: webpack -w& +. Run the following to start your app:npm start +. Test you app in the web browswer: http://localhost:3000 +. Make changes to your react component in components/App.js, e.g.: replace "Hey there, don't you love {this.state.title}" with "Hey there, don't you love {this.state.title} - totally!"
